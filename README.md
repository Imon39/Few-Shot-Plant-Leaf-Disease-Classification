# Few-Shot Learning-Based Plant Leaf Disease Classification 🌱

This project presents a lightweight few-shot learning framework for plant disease classification using a **Prototypical Network with a custom SqueezeNet backbone**.

## 📌 Overview

Plant diseases significantly impact agricultural productivity. This work focuses on solving the **limited data problem** using few-shot learning techniques.

- 🔍 Model: SqueezeNet + Prototypical Network
- ⚡ Parameters: Only **73,032**
- 📊 Max Accuracy: **95.63%**
- 🌾 Crops: Potato, Mango, Rice, Apple, Bell Pepper

## 🧠 Key Features

- Few-shot learning (N-way K-shot)
- Lightweight deep learning model
- Suitable for edge devices
- Efficient feature embedding using ProtoNet

## 📊 Results

| Plant        | Accuracy (%) |
|-------------|-------------|
| Potato      | 94.41       |
| Mango       | 52.42       |
| Rice        | 81.55       |
| Apple       | 61.21       |
| Bell Pepper | 95.63       |
